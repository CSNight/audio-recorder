## exportPCM/exportWAV 移除 + 统一 exportEncoded 导出面

- 目标：
  - 去除 `RecorderController` 上的 `exportPCM()` / `exportWAV()` 便捷方法，改为统一通过 `exportEncoded(type, options)` 调用；
  - 同时去除 `audio-recorder/encoders/pcm` 与 `audio-recorder/encoders/wav` 两个公开 subpath export，使 pcm/wav 编码器退回为纯内部实现细节。
- 完成：
  - `src/core/recorder-controller.ts`：
    - 移除 `exportPCM()` 和 `exportWAV()` 方法及其对应的类型导入（`PcmExportOptions`、`PcmExportResult`、`WavExportOptions`、`WavExportResult`）。
    - 将 `exportEncoded()` 替换为基于 `EncoderMap` 的类型安全重载版本：
      ```ts
      exportEncoded<TKey extends keyof EncoderMap>(type: TKey, options?: EncoderMap[TKey]["options"]): Promise<EncoderMap[TKey]["result"]>
      exportEncoded<TOptions, TResult>(type: string, options?: TOptions): Promise<TResult>
      exportEncoded(type: string, options?: unknown): Promise<unknown>
      ```
  - `src/index.ts`：移除 `exportPcmSnapshot` / `exportWavSnapshot` 及其类型的根入口 re-export。
  - `vite.config.ts`：移除 `encoders/pcm/index` 和 `encoders/wav/index` entry。
  - `package.json`：移除 `./encoders/pcm` 和 `./encoders/wav` export block。
  - `scripts/verify-exports.mjs`：移除 pcm/wav 的文件存在性检查和 export 结构验证。
  - `playground/app.js`：`stopRecorder()` 中三处导出调用统一改为 `recorder.exportEncoded("pcm"|"wav"|"mp3")`。
  - `tests/unit/recorder-controller.spec.ts`：全部 `exportPCM()`/`exportWAV()` 调用替换为 `exportEncoded("pcm"|"wav", options)`。
- 测试结果：
  - `npm run test:unit` 通过，144 个用例全部通过。
  - `npm run build` 通过。
  - `npm run verify:exports` 通过。
- 说明：
  - `src/encoders/pcm/` 和 `src/encoders/wav/` 目录及其源文件保留（供 `createDefaultEncoderRegistry()` 内部 import），仅移除对外 subpath。
  - `EncoderMap` 接口与 `EncoderRegistry` 内部实现不受影响，类型推断链路完整。

## lamejs MPEGMode 运行时错误修复

- 问题：`MPEGMode is not defined` 运行时报错。
  - 根因：lamejs npm 入口 (`src/js/index.js`) 使用 CJS `require('./MPEGMode.js')` 链式加载；Vite 打包后 `MPEGMode` 在浏览器中为 `undefined`。
- 修复方案：vendor `lame.all.js`（lamejs 自带的自包含 IIFE bundle）。
  - 将 `node_modules/lamejs/lame.all.js` 复制到 `src/codecs/mp3/vendor/lame.all.js`。
  - 在末尾将 `lamejs()` 改为捕获返回值，并在函数末尾添加 `return lamejs`，追加 ES module export：
    ```js
    const __lamejsExports = lamejs()
    export const Mp3Encoder = __lamejsExports.Mp3Encoder
    export const WavHeader = __lamejsExports.WavHeader
    ```
  - 添加 `src/codecs/mp3/vendor/lame.all.d.ts` 类型声明。
  - `mp3-snapshot-exporter.ts` 和 `mp3-chunked-encoder.ts` 中 `import { Mp3Encoder } from "lamejs"` 改为 `import { Mp3Encoder } from "./vendor/lame.all.js"`。
- 变更文件：
  - `src/codecs/mp3/vendor/lame.all.js`（新增）
  - `src/codecs/mp3/vendor/lame.all.d.ts`（新增）
  - `src/codecs/mp3/mp3-snapshot-exporter.ts`
  - `src/codecs/mp3/mp3-chunked-encoder.ts`
- 结果：`npm run build` 全量通过，无 TypeScript 错误。
