说明

  • 当前页面直接 import /dist/index.js，用于验证打包产物对外行为。
  • 实时导出、实时播放、ASR 输出当前仅演示 PCM/WAV 路径，其他编码仅保留 snapshot 导出。
  • Vue 通过 CDN 版 ESM 引入，不在仓库里再建一套 Vue 工程。
  • 建议先使用“外部音调流”，这样不依赖麦克风权限也能完整走一遍主链路。
  • 存储模式可切到 IndexedDB / OPFS，用于验证长录音溢写能力是否按 session 清理。
